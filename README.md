# 23 February Greeting Page

Интерактивная веб-страница для поздравления с 23 февраля на чистом HTML, CSS и JavaScript.

## Функциональность

### 🎨 Интерактивные элементы
- **Заголовок** — меняет цвет при клике
- **Танк** — корпус реагирует на наведение, башня меняет цвет случайным образом
- **Дата** — автоматически отображает текущую дату

### 📝 Персональное поздравление
- Форма с вводом имени
- Выбор рода войск из выпадающего списка (танкист, летчик, моряк, десантник, пехотинец, пограничник, ракетчик)
- Персональное обращение с учетом рода войск
- Временное изменение фона при открытии поздравления

### 🖼️ Фотогалерея
- Четыре кнопки для переключения фото
- Эффект увеличения при наведении

### 🎞️ Слайдер
- Автоматическое перелистывание по кнопке "Запустить слайды"
- Остановка по кнопке "Стоп"
- Ручное переключение двойным кликом
- Интервал смены 1.5 секунды

### 🎁 Открытка
- Отображение персонализированного поздравления
- Звезда меняет цвет при клике
- Кнопка закрытия

## Важное примечание об изображениях

Изображения взяты из открытых источников в интернете и используются **исключительно в ознакомительных и демонстрационных целях**. Все права на изображения принадлежат их законным владельцам. При коммерческом использовании необходимо заменить их на собственные или лицензионные материалы.

## Установка и запуск

1. Клонировать репозиторий
2. Добавить свои изображения в папку img/ (при необходимости)
3. Открыть index.html в любом современном браузере

## Требования

- Любой современный браузер (Chrome, Firefox, Safari, Edge)
